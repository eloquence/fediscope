FediScope lets you find fediverse accounts for people in a field using Wikidata. You can then compile a custom CSV and import it into Mastodon. It was made by @eloquence@social.coop (code | toggle dark mode).

Privacy: This tool directly interacts with Wikidata from your browser and does not log your queries. Wikidata creates records for people the community considers notable. If they made an entry for you and you don't want them to, you can request deletion there. If you're in Wikidata but would prefer not to be visible only in this tool, you can also ping me to be added to a deny-list.

This tool uses an inclusive search, so someone classified as an opinion journalist is also counted as a journalist, someone classified as a beekeper is counted an agricultural worker, and so on, depending on how Wikidata organizes its taxonomy. It includes both present and past occupations.

Select a field to begin. The number is the approximate number of accounts in that field as of a last batch analysis (December 21, 2022). Actual results may vary.
